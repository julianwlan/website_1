Moist Chocolate Cake

Ingridients


  • 200 g (7 oz) dark chocolate
  • 150 g (⅔ cup) butter
  • 150 g (¾ cup) sugar
  • 3 eggs
  • 150 g (1¼ cups) all-purpose flour
  • 1 tsp baking powder
  • 1 pinch of salt
  • 2 tbsp unsweetened cocoa powder
  • 100 ml (½ cup) milk or buttermilk
  • Optional: 1 tsp vanilla extract or 1 tsp espresso (enhances the chocolate flavor)


Instructions


  1. Preheat the oven to 180 °C / 350 °F (or 160 °C / 320 °F fan) and prepare your baking pan by greasing it
    or lining it with parchment paper.
  2. Melt the chocolate and butter together in a heatproof bowl over simmering water or in short intervals
    in the microwave, then let it cool slightly.
  3. In a large bowl, beat the eggs and sugar until the mixture becomes light and creamy.
  4. Gently stir in the melted chocolate-butter mixture until smooth and well combined.
  5. In a separate bowl, mix the flour, baking powder, salt, and cocoa powder together.
  6. Add the dry ingredients alternately with the milk to the chocolate mixture, stirring only until
    everything is just combined.
  7. Pour the batter into the prepared pan and smooth the top.
  8. Bake for 35-40 minutes, checking with a toothpick that should come out slightly moist.
  9. While still warm, drizzle 2-3 tablespoons of warm milk over the cake to make it extra moist.
  10. Let the cake cool completely, then optionally top it with melted chocolate, glaze, or powdered sugar.

Bubble Sort


            public class BubbleSort {
                public static void main(String[] args) {
                    int[] array = {5, 4, 3, 2, 1};
                    int sizeOfArray = 4;
                    boolean flag = true;
                    for(int i = 0; i <= sizeOfArray; i++) { 
                        System.out.print(array[i]);
                    }
                    System.out.println("\n");
                    
                    while(flag) {
                        int halpy = 0;
                        for(int i = 0; i < sizeOfArray; i++) {
                            int internalMovements = 0;
                            if(array[i] > array[i + 1]) {
                                internalMovements++;
                                halpy = array[i];
                                array[i] = array[i + 1];
                                array[i + 1] = halpy;
                            }
                            for(int j = 0; j <= sizeOfArray; j++) {
                                System.out.print(array[j]);
                            }
                            System.out.println("\n");
                            if(internalMovements == 0) {
                                flag = false;
                            }
                        }
                    }
                }
            }
